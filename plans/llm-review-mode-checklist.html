Design summary: When llmAutoPaste = false, after generation completes the overlay enters a new .llmReview state (amber accent, pulsing indicator, hint text "↵ paste · ESC dismiss"). Press Return to inject, ESC to discard, or wait 15s for auto-dismiss. The CGEventTap consumes the Return keypress only while review state is active.
